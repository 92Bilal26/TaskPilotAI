================================================================================
                        AUTHENTICATION FLOW
================================================================================

SIGN UP:
  1. User enters email, name, password
  2. Frontend sends to POST /auth/signup
  3. Backend hashes password with bcrypt
  4. User created in database
  5. JWT tokens returned
  6. Stored in localStorage
  7. Redirect to dashboard

SIGN IN:
  1. User enters email, password
  2. Frontend sends to POST /auth/signin
  3. Backend verifies password
  4. JWT tokens returned
  5. Stored in localStorage
  6. Redirect to dashboard

TOKEN REFRESH:
  1. Access tokens expire after 7 days
  2. Use refresh token to get new access token
  3. POST /auth/refresh endpoint
