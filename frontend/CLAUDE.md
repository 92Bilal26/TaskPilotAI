# TaskPilotAI Frontend Guidelines

## Technology Stack
- **Framework**: Next.js 16+ with React 19
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Authentication**: Better Auth SDK
- **Testing**: Jest with 90% coverage requirement

## Project Structure
- `/frontend/app/` - Next.js App Router pages
- `/frontend/components/` - React components
- `/frontend/lib/` - Utility functions and API client
- `/frontend/types/` - TypeScript interfaces
- `/frontend/tests/` - Component and integration tests

## Testing Requirements
- All components must have unit tests
- API integration must be tested
- Authentication flow must be verified
- Minimum 90% code coverage required
